# miraktest-nico

[ニコニコ実況](https://jk.nicovideo.jp) との連携を行うプラグインです。現在は録画のみ対応です。<br />
過去ログは[ニコニコ実況 過去ログ API](https://jikkyo.tsukumijima.net/)から取得します。<br />
[miraktest-dplayer](../miraktest-dplayer) か [miraktest-zenza](../miraktest-zenza) が必要です。
