# miraktest-miyou

[search-future/miyou.tv](https://github.com/search-future/miyou.tv) で使用されている miteru との連携を行うプラグインです。<br />
コメントの取得にモリタポアカウントが必要です。設定画面から指定してください。<br />
[miraktest-dplayer](../miraktest-dplayer) など対応するコメントレンダラープラグインが必要です。
